Coruse Catalog Description:

Algorithms and Problem Solving Basic Algorithmic Analysis; Advanced Algorithmic Analysis; Advanced Data Structures Algorithms Strategies & Analysis of Fundamental Computing Algorithms; Basic Computability; The Complexity Classes P and NP.

Pre-requisites:

ICS 202
ICS 253

Course Objectives:

To provide the students:

  1. The importance of studying the complexity of a given algorithm.
  2. Various algorithmic design techniques.
  3. Utilization of data structures and/or algorithmic design techniques in solving new problems.
  4. Basic computability concepts and the complexity classes P, NP, and NPComplete.
  5. . Some techniques to deal with hard problems.

Course Learning Outcomes:

Upon completion of the course, you should be able to:

  1. Analyze the complexity of a given algorithm.
  2. Apply classical sorting, searching, optimization and graph algorithms.
  3. Compare, contrast, and choose appropriate algorithmic design techniques to present an algorithm that solves a given problem.
  4. Able to explain NP-Completeness and deal with NP-complete problems.

Required Material

  • Lecture notes and some pointed websites
  • M. Alsuwaiyel, Algorithms, Design Techniques and Analysis, World Scientific, 1999.
